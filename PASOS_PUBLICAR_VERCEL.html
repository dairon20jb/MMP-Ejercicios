Publicar Pausas Activas en Vercel

Proyecto: CAPACITACIONES VIRTUALES/pausas-activas-web-vercel

1) Requisitos previos

  • Tener cuenta en Vercel.
  • (Opcional) Tener cuenta en GitHub para despliegue automático.
  • Usar navegador con permisos de cámara y HTTPS.

2) Opción A (rápida): drag and drop

  1. Comprime la carpeta pausas-activas-web-vercel en un .zip.
  2. Entra a Vercel y crea un proyecto nuevo.
  3. Arrastra el .zip o la carpeta según la interfaz de Vercel.
  4. Publica con la configuración por defecto (sitio estático).

3) Opción B: con GitHub (recomendada)

  1. Sube esta carpeta a un repositorio.
  2. En Vercel: Add New Project y conecta el repo.
  3. Framework preset: Other (sitio estático).
  4. Si el repo tiene más carpetas, define Root Directory como: CAPACITACIONES VIRTUALES/pausas-activas-web-vercel.
  5. Deploy.

4) Validación post-deploy

  1. Abre la URL pública y pulsa Iniciar camara.
  2. Acepta permisos de cámara del navegador.
  3. Verifica que cambie a Estado: deteccion activa al mostrar la mano.
  4. Comprueba que suben los contadores de los 4 ejercicios.

5) Problemas frecuentes y solución

  • No abre cámara: revisa permisos del sitio y que no esté ocupada por Zoom/Meet.
  • No detecta mano: mejora luz, acerca la mano y evita fondo muy oscuro.
  • No cuenta OK sostenido: mantén pinza pulgar-índice al menos 1.0 s y luego suelta.
  • Carga lenta inicial: normal, el modelo se descarga la primera vez.
